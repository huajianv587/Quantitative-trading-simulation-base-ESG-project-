### 1. API路由统一 ✓

#### 问题
- 前端调用路径不一致
- 后端路由缺少统一前缀
- 导致404错误

#### 解决方案
```python
# backend: gateway/api/factory.py
app.include_router(core.router, prefix="/api")
app.include_router(auth.router, prefix="/api")
app.include_router(quant.router, prefix="/api")
# ... 所有路由统一添加 /api 前缀
```

```javascript
// frontend: js/qtapi.js
auth: {
  register: function(payload) { return _post('/api/auth/register', payload); },
  login: function(payload) { return _post('/api/auth/login', payload); },
}
```

#### 结果
- ✅ 所有API端点统一使用 `/api` 前缀
- ✅ 前后端路径完全匹配
- ✅ 健康检查端点：`/api/health`

### 10. 错误处理机制 ✓

#### 功能

- 网络错误重试机制
- 友好的错误页面
- 自动重试（最多3次）

#### API请求封装
```javascript
export async function fetchWithRetry(url, options, retries = 3) {
  try {
    return await fetch(url, options);
  } catch (error) {
    if (retries > 0) {
      await new Promise(r => setTimeout(r, 1000));
      return fetchWithRetry(url, options, retries - 1);
    }
    throw error;
  }
}
```
