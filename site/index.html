Architecture Layers

按 8 个垂直层组织系统，而不是堆功能点。

平台当前按 L0-L8 组织，便于后续继续接卫星数据、RL Agent、实验追踪和云端训练。

L0-L1

数据接入 + 治理

市场、宏观、ESG、另类数据进入统一时序治理和血缘追踪流程。

L2-L4

分析引擎 + Agent

技术指标、因子工程、ESG RAG、Research Agent、Strategy Agent 和 Risk Agent 协同工作。

L5-L6

风控 + 回测执行

回撤、波动、CVaR、样本外验证、Paper Trading 和交易成本估计形成闭环。

L7-L8

实验 + 展示交付

实验记录、工件留存、产品站、报告中心和研究数据出口统一沉淀。
