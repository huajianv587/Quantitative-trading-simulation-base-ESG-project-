# 后端异步化改造报告

## 改造概述

将后端 API 路由从同步改为异步，提升并发处理能力和系统吞吐量。

**改造时间**: 2026-04-21  
**改造范围**: Gateway API 路由层 + 数据库客户端层  
**预期提升**: 吞吐量提升 3-5 倍，响应时间降低 40-60%

---

## 改造内容

### 1. 数据库客户端层异步化

#### 新建异步数据库客户端
**文件**: `gateway/db/async_client.py`

**核心功能**:
- 使用 `supabase-py` 异步客户端 (`acreate_client`)
- 实现异步版本的所有数据库操作函数
- 使用 `asyncio.Lock` 替代 `threading.Lock` 确保线程安全
- 支持连接池和超时配置

**主要函数**:
```python
async def get_client() -> AsyncClient
async def create_session_async(session_id: str, user_id: str | None = None)
async def save_message_async(session_id: str, role: str, content: str)
async def get_history_async(session_id: str, limit: int = 20)
async def save_query_history_async(...)
async def get_query_history_async(...)
```

### 2. API 路由层异步化

#### 2.1 Agent 路由 (`gateway/api/routers/agent.py`)

**改造端点**:
- ✅ `POST /session` - 会话创建
- ✅ `POST /query` - 查询处理（含历史记录检索）
- ✅ `GET /history/{session_id}` - 历史记录获取
- ✅ `POST /agent/analyze` - ESG 分析
- ✅ `POST /agent/esg-score` - ESG 评分

**改造要点**:
```python
# 改造前
@router.post("/session")
def new_session(session_id: str, user_id: str | None = None):
    runtime.create_session(session_id=session_id, user_id=user_id)
    return {"session_id": session_id, "created": True}

# 改造后
@router.post("/session")
async def new_session(session_id: str, user_id: str | None = None):
    from gateway.db.async_client import create_session_async
    await create_session_async(session_id=session_id, user_id=user_id)
    return {"session_id": session_id, "created": True}
```

#### 2.3 User 路由 (`gateway/api/routers/user.py`)

**改造端点**:
- ✅ `POST /user/reports/subscribe` - 报告订阅
- ✅ `GET /user/reports/subscriptions` - 获取订阅列表
- ✅ `PUT /user/reports/subscriptions/{subscription_id}` - 更新订阅
- ✅ `DELETE /user/reports/subscriptions/{subscription_id}` - 取消订阅

**改造策略**:
使用 `loop.run_in_executor` 将同步数据库操作包装为异步：
```python
import asyncio
loop = asyncio.get_event_loop()

def _fetch():
    return runtime.get_client().table("...").select("*").execute().data

rows = await loop.run_in_executor(None, _fetch)
```

#### 2.4 Auth 路由 (`gateway/api/routers/auth.py`)

- ✅ `POST /auth/reset-password/request` - 密码重置请求
- ✅ `POST /auth/reset-password/confirm` - 密码重置确认
- ✅ `GET /auth/verify` - Token 验证

```python
user = await loop.run_in_executor(None, _repo.get_user_by_email, email)
await loop.run_in_executor(None, _repo.record_auth_audit, "login", email, user_id, True, "success")
```

## 技术要点

### 1. 异步数据库客户端
```python
from supabase import acreate_client
import asyncio

_client: AsyncClient | None = None
_lock = asyncio.Lock()

async def get_client() -> AsyncClient:
    global _client
    if _client is None:
        async with _lock:
            if _client is None:
                _client = await acreate_client(url, key)
    return _client
```

### 2. 同步代码异步包装
对于无法直接改为异步的同步代码（如 repository 层），使用 `run_in_executor`：
```python
import asyncio
loop = asyncio.get_event_loop()
result = await loop.run_in_executor(None, sync_function, arg1, arg2)
```

### 3. 异步函数签名
```python
# 改造前
def endpoint():
    pass

# 改造后
async def endpoint():
    pass
```
