**Windows:**
```bash
start_services.bat
```

**Linux/Mac:**
```bash
chmod +x start_services.sh
./start_services.sh
```

### 2. 访问应用

- 前端: http://localhost:8080
- 后端: http://localhost:8012
- API文档: http://localhost:8012/docs

### 3. 运行测试

```bash
python test_complete.py
```
