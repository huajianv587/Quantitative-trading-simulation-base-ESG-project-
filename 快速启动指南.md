# 🚀 Quant Terminal - 快速启动指南

## 📋 前置要求

- Python 3.8+
- 已安装依赖包（见 requirements.txt）
- 端口 8000 和 8080 可用

---

## ⚡ 快速启动

### Windows 用户

双击运行：
```
start_services.bat
```

### Linux/Mac 用户

```bash
chmod +x start_services.sh
./start_services.sh
```

---

## 🌐 访问地址

启动成功后，访问以下地址：

- **前端应用**: http://localhost:8080
- **后端API**: http://localhost:8000
- **API文档**: http://localhost:8000/docs

## 📝 重要提示

### ⚠️ 首次启动

1. 确保已安装所有依赖：
```bash
cd full_suite_source_bundle_20260421
pip install -r requirements.txt
```

2. 复制环境变量文件：
```bash
cp .env.example .env
```

3. 根据需要修改 `.env` 配置

### ⚠️ 代码修改后

如果修改了后端代码，需要重启服务器：
- Windows: 关闭命令窗口，重新运行 `start_services.bat`
- Linux/Mac: 按 Ctrl+C 停止，重新运行 `./start_services.sh`

## 🐛 故障排除

### 问题1: 端口被占用

**错误**: `Address already in use`

**解决**:
```bash
# Windows
netstat -ano | findstr :8000
taskkill /PID <PID> /F

# Linux/Mac
lsof -ti:8000 | xargs kill -9
```

### 问题2: 404错误

**原因**: 后端服务未启动或路由配置未生效

**解决**:
1. 确认后端服务正在运行
2. 检查 http://localhost:8000/docs 是否可访问
3. 重启后端服务

### 问题3: CORS错误

**原因**: 跨域配置问题

**解决**:
检查 `.env` 文件中的 `CORS_ORIGINS` 配置
